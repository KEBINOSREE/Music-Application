A responsive web-based music application that allows users to stream songs and manage their personal accounts securely. The application focuses on authentication, user profile management, and smooth music playback experience.
The project is deployed using Firebase Hosting for reliable and secure access.

Key Features:
🎶 Music streaming functionality
🔐 User Authentication (Register / Login / Logout)
👤 Update profile information
🖼 Update profile picture
🔑 Change password
❌ Delete account
📱 Fully responsive design
🌐 Deployed using Firebase Hosting

Tech Stack:
- Frontend: React.js  
- Styling: Tailwind CSS  
- Routing: React Router DOM  
- Authentication: Firebase Authentication  
- Hosting: Firebase Hosting  
- Build Tool: Vite

This project was developed to gain hands-on experience with:

* Account management functionality
* Secure password handling

* Building responsive UI applications
